Help & Feedback

Pan works by analyzing the pages that you visit for sources, then replacing mentions of those sources with links to their Wikipedia page. For privacy (and performance) reasons, Pan will only monitor the pages that you tell it to. In order to add a website to this list, simply click the "Add Current Site" button in the main menu. You can remove sites through the "Manage All Sites" button.

Pan is made by a single developer, and uses a number of different technologies. To learn more information about Pan, the source code, or its creator, you can visit GitHub.

Pan takes data privacy very seriously. The information collected by our server is simply the article URL and the content of the page. This information is not tied back to you, as a user, in any form, nor is this data shared with anyone. If you are concerned with the accuracy of these statements, you can contact the creator via the Support and Feedback option.

Since Pan was made by a singular developer, support and updates might be slow. If you'd like to contact the creator for feedback or support, you can send an email. If you'd like to contribute to the project, you can visit the creator on GitHub.
